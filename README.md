# Personal Portfolio Website

## Overview

This project is a responsive personal portfolio website built using **HTML5**, **CSS3**, and **Bootstrap 5**. It showcases my skills, projects, and contact information in a clean and modern layout. The website is fully responsive and adapts to different screen sizes, providing a user-friendly experience on desktops, tablets, and mobile devices.

---

## Features

- Responsive navigation bar
- Full-screen hero section
- About Me section
- Projects showcase
- Contact form
- Social media links
- Responsive design using Bootstrap 5
- Clean and modern user interface

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension (optional).
4. Open `index.html`.
5. Run the project using Live Server or open the file directly in your browser.

## Website Sections

### Home
Introduces the portfolio with a full-screen hero section displaying the developer's name and professional role.

### About Me
Provides information about the developer, including skills, profile, and contact details.

### Projects
